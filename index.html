📦 نصب برای دبیان/اوبونتو

برای نصب NotiSync در سیستم‌های مبتنی بر دبیان (شامل اوبونتو، مینت، و ...) مراحل زیر را دنبال کنید:

مرحله 1: دانلود فایل .deb

wget https://golgavzabun.storage.c2.liara.space/notiSync/notisync_1.0.0-1_amd64.deb

مرحله 2: نصب با dpkg

sudo dpkg -i notisync_1.0.0-1_amd64.deb

مرحله 3: رفع وابستگی‌ها (در صورت نیاز)

sudo apt-get install -f

مرحله 4: اجرای برنامه

notisync

🔧 روش جایگزین با apt

اگر ترجیح می‌دهید از apt استفاده کنید:

نصب مستقیم از URL

sudo apt install ./notisync_1.0.0-1_amd64.deb

⚠️ نکات مهم

  • اطمینان حاصل کنید که سیستم شما 64-bit است
  • برای نصب نیاز به دسترسی sudo دارید
  • در صورت بروز خطا، ابتدا وابستگی‌ها را نصب کنید
  • برنامه پس از نصب در منوی برنامه‌ها قابل دسترسی است

📱 مراحل نصب در macOS

برای نصب NotiSync در سیستم‌های macOS مراحل زیر را دنبال کنید:

1

بعد از اینکه برروی NotiSync.dmg کلیک کرده و در پنجره باز شده، برنامه را به پوشه Application منتقل کردید. برروی آیکون برنامه در لیست برنامه‌های نصب شده دستگاه کلیک کنید.

2

یک پنجره راجع به ناشناس بودن توسعه‌دهنده نمایش داده می‌شود که بدلیل تحریم‌های توسعه‌دهندگان ایرانی امکان حل این مشکل نیست، پس لازم هست دفعه اول مراحل زیر را پیش ببرید، برروی Done بزنید.

3

وارد تنظیمات سیستم عامل شده و در پایین بخش Privacy & Security برروی Open Anyway کلیک کنید. در پنجره باز شده مجدد برروی Open Anyway کلیک کرده رمز خود را وارد نمایید. (اگر کیبورد شما اثر انگشت دارد می‌توانید اثرانگشت بزنید.)

4

حال می‌توانید بدون مشکل از برنامه استفاده نمایید. 🎉

✅ برنامه با موفقیت نصب شد!

⚠️ نکات مهم

  • اطمینان حاصل کنید که سیستم شما macOS 10.14 یا بالاتر است
  • برای نصب نیاز به دسترسی مدیر دارید
  • در صورت بروز خطا، ابتدا تنظیمات امنیتی را بررسی کنید
  • برنامه پس از نصب در Launchpad قابل دسترسی است
